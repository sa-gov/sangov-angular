GETTING STARTED

To get started you need to download our State Template from GitHub (TBA). Also, you can refer to our Getting Started page.
  1. Core css and colorscheme files from /css folder (see our update page CSS)
  2. Core js files and other js libraries from /js folder (see our update page)
  3. Fonts files from /fonts folder
  4. Structure includes in /ssi folder (see our structure page)

SSI must be set up on your server for them to work. You are unlikely to have SSI enabled if you do not have a CGI-BIN but if you do it is very likely that the server administrator will have enabled SSI.

INSTALLATION

  1. Go to State Template GitHub repository (TBA) page and click on "clone or download" button, then copy GitHub repository web URL
  2. Open your Visual Studio and go to the Team Explorer. Make sure your Visual Studio has GitHub extension.
  3. Click on the "Manage Connections" icon then in your GitHub connections section click on "clone" link. That will open up the "Open from GitHub" popup.
  4. Paste in State template GitHub repository web URL and click "Clone". That will clone the State Template from our GitHub repository to your local machine. Once it's cloned you can open the cloned website from your Visual Studio.
